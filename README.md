# MediConnect - Healthcare Platform

MediConnect is a next-generation healthcare platform that connects patients with doctors, manages prescriptions, and provides comprehensive healthcare management.

## Features

- **Video Consultations**: Connect with doctors instantly through HD video calls.
- **Digital Prescriptions**: Receive and manage prescriptions electronically.
- **Secure & Private**: Your health data is encrypted and protected.
- **24/7 Available**: Access healthcare services anytime, anywhere.
- **Pharmacy Integration**: Manage inventory and process prescriptions.

## Technologies Used

- **Vite**: Next-generation frontend tool.
- **TypeScript**: Typed JavaScript for better development experience.
- **React**: Modern UI library.
- **shadcn-ui**: High-quality UI components.
- **Tailwind CSS**: Utility-first CSS framework.
- **Supabase**: Backend-as-a-service for authentication and database.

## Getting Started

### Prerequisites

- Node.js & npm installed.

### Installation

1.  **Clone the repository**:
    ```sh
    git clone <YOUR_GIT_URL>
    ```

2.  **Navigate to the project directory**:
    ```sh
    cd health-harmony
    ```

3.  **Install dependencies**:
    ```sh
    npm install
    ```

4.  **Start the development server**:
    ```sh
    npm run dev
    ```

## Development

- `npm run dev`: Start the development server.
- `npm run build`: Build the project for production.
- `npm run lint`: Run ESLint to check for code quality.
- `npm run preview`: Preview the production build locally.
